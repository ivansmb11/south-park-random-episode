# Discovery Answers

## Q1: Should we use Tailwind CSS v4 as required by SHADCN Vue, replacing the current custom CSS styling?
**Answer:** Yes

## Q2: Should we maintain the current dark color scheme or adopt SHADCN's default light theme?
**Answer:** Make it able to switch between dark and light mode

## Q3: Should we create a separate LayoutComponent as a dedicated component or integrate it into the existing App.vue structure?
**Answer:** Separate LayoutComponent (better separation of concerns and reusability)

## Q4: Should we migrate all existing custom-styled components (buttons, cards, inputs) to use SHADCN components?
**Answer:** Yes

## Q5: Should we add additional SHADCN components like Dialog, Sheet, or Toast for enhanced UX beyond the current functionality?
**Answer:** No
